# image-to-text-app
A web-based OCR (Optical Character Recognition) application built with Django that extracts text from images and translates it into multiple languages.
# 🧠 OCR + Multilingual Text Translator

A Django-based web application that extracts text from images using OCR and translates it into 20+ languages.

---

## 🚀 Features

- 📷 Image upload support
- 🔍 Text extraction using OCR (Tesseract)
- 🌐 Translation into 20+ languages
- 📋 Copy to clipboard feature
- 🎨 Clean and responsive UI
- ⚡ Fast processing

---

## 🛠️ Tech Stack

- Python
- Django
- Tesseract OCR
- Deep Translator / Google Translate API
- HTML, CSS, JavaScript
